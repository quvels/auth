

# Quvels Auth microservice

  

## Description

  

Auth microservice

  

## Installation


Rename **.env.example** file to **.env**
  

```bash

$ npm install

```

  

## Running the app

  

```bash

# Docker development

docker-compose -f docker-compose.yml -f docker-compose.debug.yml up

```

  

## Test

  

```bash

# unit tests

$ npm run test

  

# e2e tests

$ npm run test:e2e

  

# test coverage

$ npm run test:cov

```
